## 技術要點（教練叮嚀）

### 深蹲
- 站距偏窄要修正，腳尖與膝蓋方向一致
- 足底三點平衡，重心不可往內倒（會造成腳跟飄起 / 夾腳）
- 呼吸不要太趕
- 4 組重量只能微調一次；多次微調代表前面幾組不算訓練組

### 相撲硬舉 T&G
- 準備姿勢臀位不能太低，T&G 觸地瞬間立刻離地
- 下放時不可讓槓繞膝蓋（需推髖，而非上半身前傾）
- 家中槓片直徑較小 → **一定要用墊子**
- 站距約兩倍肩寬

### 反式划船
- 錄影請從 45 度角，方便看胸口與髖是否挺好
- 架設高度保持一致（比對上次）

### 單邊啞鈴臥推
- 頭要有一半露出椅子外
- 腳放坐墊側，不要踩到頭枕位置

### 跪姿肩推
- 左右腳在後時，腳尖都要撐地
- 換腳前先確認啞鈴已放下

### 滑輪下拉
- 拉至下巴即可，手肘朝下
- 放回時手要完全伸直才算完成，不可拉著站起來

### 酒杯跨步蹲
- 啞鈴「捧」在下巴下方，不是「提」
- 後跨時重心留在原地，後腳只是去點地，不要把重心跟著往後壓
- 收腳立正時注意不外八

### 側棒式
- 上手插腰，手肘朝天
- 不低頭，頭保持正中
- 屁股要夠離地

### 懸吊舉腿
- 盡量只微屈膝（舉腿，不是舉膝）
- 不可舉到最後變聳肩
